Background:

I have always been fascinated by technology, whether it was how remote control cars worked or how they would make cartoons and animated movies; these kinds of stuff really intrigued me. I think the first sign to why I chose this profession started showing in Grade 9. I was on the robotics team and we had to assemble a robot that would perform certain tasks. Assembling the robot was cool but I was blown away by the software side of it. I helped write code that made the robot move depending on remote control actions. So, writing two lines of code that moved the robot forward blew my mind. I finally understood how a remote control car worked and being able to create something similar to my favorite childhood toy was incredible.

Being on the robotics team was one of the best experiences I've had. It was a blast working with my teammates and we got the opportunity to go Orlando, Florida to compete in the Vex World Championship 2012.

It was in grade 10 where I got to see more of the software side when I had to build a game in Flash Action Script. This was where I learned more about computer programming and got a better idea of what it can really do. View Game

Space Run - Android SideScroller

I created this as final project to grade 12 programming class. I used a library called LibGDX that does the heavy lifting such as creating vectors and such. This was all coded in Java.


This app is currently not on the playstore, however, you can download the apk from here and put it on your android phone if you wish.

Download Android APK of the game here

Steps to install APK to phone

  • 1. Copy the APK to your phone.
  • 2. If you don't have a FileManager, download one. I use Astro File Manager (its free)
  • 3. Go to Settings>Security and check the "Unknown Sources" box
  • 4. Locate the APK file and click on it to install it.
  • 5. Enjoy

You can play the game on your desktop if you want.
