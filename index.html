Space Run - Android SideScroller

I created this as final project to grade 12 programming class. I used a library called LibGDX that does the heavy lifting such as creating vectors and such. This was all coded in Java.


This app is currently not on the playstore, however, you can download the apk from here and put it on your android phone if you wish.

Download Android APK of the game here

Steps to install APK to phone

  • 1. Copy the APK to your phone.
  • 2. If you don't have a FileManager, download one. I use Astro File Manager (its free)
  • 3. Go to Settings>Security and check the "Unknown Sources" box
  • 4. Locate the APK file and click on it to install it.
  • 5. Enjoy

You can play the game on your desktop if you want.
