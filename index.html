Slack Translate
Slack Translate is a Slack command which takes in a word or phrase and spits out the translations in French, Spanish and Portuguese. During my coop term at QuickTapSurvey, we supported 4 languages; EN, ES, FR, PT, thus localizing a word or phrase into four different languages. The current way this was done was going on Google Translate and typing the word or phrase and selecting the languages you want to translate. This caught my attention as this was a very inefficient way and it slowed down development time. Since we used Slack as our primary communication, I decided to create a Slack command to make this task more efficient.

Since we kept a collection of phrases we localized, this command first looks at our localization db and if the phrase is not found or a similar match is not found, it will use an online translation API.

Tech Stack: NodeJS, SlackAPI, MicrosoftTranslateAPI
